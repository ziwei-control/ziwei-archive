## 任务名称
Python 天气查询工具

## 任务描述
创建一个 Python 命令行工具，可以查询全球任意城市的实时天气。

## 功能需求

### 核心功能
1. 查询城市当前天气（温度、湿度、风速）
2. 支持城市名称搜索
3. 显示天气预报（3 天）
4. 支持摄氏/华氏温度切换

### 高级功能
1. 自动定位当前城市
2. 天气警报推送
3. 历史天气查询
4. 多城市对比

### 技术栈
- Python 3.8+
- 使用 OpenWeatherMap API
- 使用 argparse 处理命令行
- 使用 json 处理数据

### 安全要求
1. API Key 安全存储
2. 请求频率限制
3. 错误处理完善

## 输出要求
- 可执行 Python 脚本
- README 文档
- 单元测试
- API 配置示例
