# AI 辅助编程助手 - 完整实现说明书

**版本**: v5.0

## 1. 产品概述

AI 驱动的编程助手，提供代码片段管理、自动生成注释、批量重命名等功能。

## 2. 核心功能

### 2.1 代码片段管理
- 保存常用代码
- 快速搜索
- 分类管理

### 2.2 自动生成注释
- 函数注释
- 类注释
- 模块注释

### 2.3 批量重命名
- 变量重命名
- 函数重命名
- 文件重命名

## 3. 安装使用

```bash
# 安装
pip install -r requirements.txt

# 使用
python3 src/main.py --help
```

## 4. 功能详解

### 4.1 代码生成
```python
from ai_model_caller import AIModelCaller

ai = AIModelCaller()
code = ai.generate_code('python', 'sort function')
```

### 4.2 代码审查
```python
from code_reviewer import CodeReviewer

reviewer = CodeReviewer()
issues = reviewer.review(code)
```

## 5. 使用流程

```
输入需求 → AI 理解 → 生成代码 → 审查优化 → 输出结果
```
