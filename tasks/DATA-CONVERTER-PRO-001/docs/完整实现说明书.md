# 专业数据转换工具 - 完整实现说明书

**版本**: v5.0

## 1. 产品概述

支持 CSV、JSON、XML、Excel 等多种格式的数据转换工具。

## 2. 支持格式

| 格式 | 输入 | 输出 |
|------|------|------|
| CSV | ✅ | ✅ |
| JSON | ✅ | ✅ |
| XML | ✅ | ✅ |
| Excel | ✅ | ✅ |
| YAML | ✅ | ✅ |

## 3. 安装使用

```bash
pip install pandas openpyxl xmltodict
python3 src/main.py convert input.csv output.json
```

## 4. 核心功能

### 4.1 批量转换
```bash
python3 src/main.py batch-convert ./input/ ./output/ --from csv --to json
```

### 4.2 字段映射
```bash
python3 src/main.py convert input.csv output.json --mapping old_name:new_name
```

### 4.3 数据验证
```bash
python3 src/main.py validate data.json --schema schema.json
```

## 5. 使用示例

**CSV 转 JSON**:
```bash
python3 src/main.py convert data.csv data.json
```

**JSON 转 Excel**:
```bash
python3 src/main.py convert data.json data.xlsx
```
