## 任务名称
Python 实用工具 - 文件批量重命名工具

## 任务描述
创建一个 Python 命令行工具，可以批量重命名指定目录下的文件。

## 功能需求

### 核心功能
1. 支持按前缀重命名（例如：file_001.txt, file_002.txt）
2. 支持按日期重命名（例如：20260304_001.txt）
3. 支持添加后缀（例如：photo_v1.jpg）
4. 支持预览改名效果（不实际执行）
5. 支持撤销操作（备份原文件名）

### 高级功能
1. 支持正则表达式匹配
2. 支持排除特定文件
3. 支持子目录递归处理
4. 支持文件类型过滤（只处理 .jpg, .png 等）

### 安全要求
1. 操作前自动备份文件列表
2. 支持 dry-run 模式（预览）
3. 避免文件名冲突
4. 操作日志记录

## 技术栈
- Python 3.8+
- 使用 argparse 处理命令行参数
- 使用 pathlib 处理文件路径
- 使用 json 保存备份信息

1. 可执行的 Python 脚本

3. 单元测试文件
