## 2. 功能详解

### **2.1 核心功能**

#### **功能 1: 多链钱包余额查询**

**支持的区块链**:
- Bitcoin (BTC)
- Ethereum (ETH)
- XRP (Ripple)
- Dogecoin (DOGE)
- 更多链可扩展

**实现原理**:
```
用户输入钱包地址
    ↓
调用对应区块链 API
    ↓
获取余额数据
    ↓
转换为 USD 价值
    ↓
显示结果
```

**代码实现** (`src/wallet_checker.py`):
```python
#!/usr/bin/env python3
"""
钱包余额查询模块
"""

import requests
from typing import Dict, Optional

class WalletChecker:
    """钱包余额检查器"""
    
    def __init__(self):
        # API 端点配置
        self.api_endpoints = {
            'BTC': 'https://api.blockcypher.com/v1/btc/main/addrs/{}/balance',
            'ETH': 'https://api.etherscan.io/api?module=account&action=balance&address={}',
            'XRP': 'https://data.ripple.com/v2/accounts/{}',
            'DOGE': 'https://chain.so/api/v2/address/balance/DOGE/{}'
        }
    
    def check_balance(self, coin: str, address: str) -> Optional[Dict]:
        """
        查询钱包余额
        
        Args:
            coin: 币种 (BTC/ETH/XRP/DOGE)
            address: 钱包地址
            
        Returns:
            余额信息字典，失败返回 None
        """
        if coin not in self.api_endpoints:
            print(f"❌ 不支持的币种：{coin}")
            return None
        
        try:
            url = self.api_endpoints[coin].format(address)
            response = requests.get(url, timeout=10)
            
            if response.status_code == 200:
                data = response.json()
                return self._parse_response(coin, data)
            else:
                print(f"❌ API 请求失败：{response.status_code}")
                return None
                
        except Exception as e:
            print(f"❌ 查询失败：{e}")
            return None
    
    def _parse_response(self, coin: str, data: Dict) -> Dict:
        """解析 API 响应"""
        if coin == 'BTC':
            balance = data.get('balance', 0) / 1e8  # 转换为 BTC
        elif coin == 'ETH':
            balance = int(data.get('result', 0)) / 1e18  # 转换为 ETH
        elif coin == 'XRP':
            balance = float(data.get('balance', 0)) / 1e6  # 转换为 XRP
        elif coin == 'DOGE':
            balance = float(data.get('data', {}).get('balance', 0))
        else:
            balance = 0
        
        return {
            'coin': coin,
            'balance': balance,
            'address': data.get('address', '')
        }


# 使用示例
if __name__ == '__main__':
    checker = WalletChecker()
    
    # 查询 BTC 钱包
    btc_address = '1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa'
    result = checker.check_balance('BTC', btc_address)
    
    if result:
        print(f"BTC 余额：{result['balance']} BTC")
```

#### **功能 2: 实时价格获取**

**数据源**: CoinGecko API（免费）

**实现流程**:
```
定时任务（每 5 分钟）
    ↓
调用 CoinGecko API
    ↓
获取价格数据
    ↓
缓存到本地
    ↓
更新显示
```

**代码实现** (`src/price_tracker.py`):
```python
#!/usr/bin/env python3
"""
价格追踪模块
"""

import requests
import time
from datetime import datetime

class PriceTracker:
    """价格追踪器"""
    
    def __init__(self):
        self.base_url = 'https://api.coingecko.com/api/v3'
        self.cache = {}
        self.cache_time = {}
        self.cache_ttl = 300  # 5 分钟缓存
    
    def get_price(self, coin_id: str, currency: str = 'usd') -> Optional[float]:
        """
        获取实时价格
        
        Args:
            coin_id: CoinGecko 币种 ID (bitcoin, ethereum, ripple)
            currency: 货币单位 (usd, cny)
            
        Returns:
            价格，失败返回 None
        """
        # 检查缓存
        cache_key = f"{coin_id}_{currency}"
        if self._is_cache_valid(cache_key):
            return self.cache[cache_key]
        
        try:
            url = f"{self.base_url}/simple/price"
            params = {
                'ids': coin_id,
                'vs_currencies': currency
            }
            
            response = requests.get(url, params=params, timeout=10)
            
            if response.status_code == 200:
                data = response.json()
                price = data.get(coin_id, {}).get(currency)
                
                # 更新缓存
                if price:
                    self.cache[cache_key] = price
                    self.cache_time[cache_key] = time.time()
                
                return price
            else:
                return None
                
        except Exception as e:
            print(f"获取价格失败：{e}")
            return None
    
    def _is_cache_valid(self, cache_key: str) -> bool:
        """检查缓存是否有效"""
        if cache_key not in self.cache_time:
            return False
        
        age = time.time() - self.cache_time[cache_key]
        return age < self.cache_ttl


# 使用示例
if __name__ == '__main__':
    tracker = PriceTracker()
    
    # 获取 BTC 价格
    btc_price = tracker.get_price('bitcoin', 'usd')
    print(f"BTC 价格：${btc_price}")
    
    # 获取 ETH 价格
    eth_price = tracker.get_price('ethereum', 'usd')
    print(f"ETH 价格：${eth_price}")
```

#### **功能 3: 资产报告生成**

**支持格式**: JSON, CSV, TXT

**报告内容**:
- 每个钱包的余额
- USD 总价值
- 占比分析
- 时间戳

**代码实现** (`src/report_generator.py`):
```python
#!/usr/bin/env python3
"""
报告生成模块
"""

import json
import csv
from datetime import datetime
from pathlib import Path

class ReportGenerator:
    """报告生成器"""
    
    def __init__(self, output_dir: str = 'reports'):
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(exist_ok=True)
    
    def generate_json(self, portfolio: Dict) -> str:
        """生成 JSON 报告"""
        timestamp = datetime.now().strftime('%Y%m%d_%H%M%S')
        filename = self.output_dir / f'portfolio_{timestamp}.json'
        
        report = {
            'timestamp': datetime.now().isoformat(),
            'total_value_usd': portfolio.get('total_value', 0),
            'wallets': portfolio.get('wallets', []),
            'summary': self._generate_summary(portfolio)
        }
        
        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(report, f, indent=2, ensure_ascii=False)
        
        return str(filename)
    
    def generate_csv(self, portfolio: Dict) -> str:
        """生成 CSV 报告"""
        timestamp = datetime.now().strftime('%Y%m%d_%H%M%S')
        filename = self.output_dir / f'portfolio_{timestamp}.csv'
        
        with open(filename, 'w', newline='', encoding='utf-8') as f:
            writer = csv.writer(f)
            
            # 表头
            writer.writerow(['币种', '钱包地址', '余额', 'USD 价值', '占比'])
            
            # 数据行
            for wallet in portfolio.get('wallets', []):
                writer.writerow([
                    wallet['coin'],
                    wallet['address'][:10] + '...',
                    wallet['balance'],
                    wallet['value_usd'],
                    f"{wallet['percentage']:.2f}%"
                ])
        
        return str(filename)
    
    def _generate_summary(self, portfolio: Dict) -> Dict:
        """生成摘要信息"""
        total = portfolio.get('total_value', 0)
        wallets = portfolio.get('wallets', [])
        
        return {
            'total_wallets': len(wallets),
            'total_value_usd': total,
            'top_coin': max(wallets, key=lambda x: x.get('value_usd', 0))['coin'] if wallets else 'N/A'
        }
```

## 8. 故障排除

### **8.1 常见问题**

**问题 1: API 请求失败**
```
❌ API 请求失败：429
```
**原因**: 请求过于频繁，触发限流  
**解决**: 
- 等待 5 分钟后重试
- 使用缓存数据
- 申请 API Key 提高限额

**问题 2: 钱包地址无效**
```
❌ 无效的钱包地址
```
**原因**: 地址格式错误  
**解决**:
- 检查地址是否正确
- 确认币种与地址匹配
- 使用区块浏览器验证

**问题 3: 依赖安装失败**
```
❌ Could not find a version that satisfies the requirement...
```
**原因**: Python 版本不兼容  
**解决**:
```bash
# 检查 Python 版本
python3 --version

# 升级到 Python 3.8+
sudo apt update
sudo apt install python3.8
```
