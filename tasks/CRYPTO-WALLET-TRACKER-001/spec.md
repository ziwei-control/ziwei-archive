# 任务说明书

## 任务 ID
CRYPTO-WALLET-TRACKER-001

## 任务名称
Crypto Portfolio Tracker - 加密货币资产追踪器

## 任务描述
创建一个命令行工具，批量查询多个区块链钱包的余额，自动计算总价值（USD），生成资产报告。

## 功能需求

### 核心功能
1. 支持多链钱包（BTC/ETH/XRP/DOGE 等）
2. 批量查询余额（支持导入地址列表）
3. 实时价格获取（CoinGecko API）
4. 自动计算总价值（USD）
5. 生成资产报告（JSON/CSV/TXT）

### 高级功能
1. 价格异常警报（涨跌超过设定百分比）
2. 历史记录保存
3. 定时自动查询
4. 支持配置文件

### 安全要求
1. 只读操作（不需要私钥）
2. API 限流处理
3. 错误重试机制

## 技术栈
- Python 3.8+
- requests 库
- CoinGecko API（免费）

## 输出要求
- 可执行 Python 脚本
- 配置文件示例
- README 文档
- 使用示例

- [ ] 余额计算准确
- [ ] 报告生成正确
- [ ] 错误处理完善
