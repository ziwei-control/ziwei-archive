# 文件批量处理大师 - 完整实现说明书

**版本**: v5.0

## 1. 产品概述

批量重命名、格式转换、压缩解压、自动分类的文件处理工具。

## 2. 核心功能

### 2.1 批量重命名
- 前缀/后缀
- 序列号
- 正则表达式

### 2.2 格式转换
- 图片格式
- 文档格式
- 音频视频

### 2.3 自动分类
- 按类型
- 按日期
- 按大小

## 3. 安装使用

```bash
pip install Pillow
python3 src/main.py --help
```

## 4. 使用示例

**批量重命名**:
```bash
python3 src/main.py rename ./photos --prefix "vacation_" --sequential
```

**格式转换**:
```bash
python3 src/main.py convert ./images --from png --to jpg
```

**自动分类**:
```bash
python3 src/main.py organize ./downloads --by type
```
