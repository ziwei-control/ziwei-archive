### **问题 1: 进程数异常**

**现象**:
```

日志修剪：3 个 (应该=1)

```

**原因**:

- 旧进程未清理

**修复**:
```bash

ps aux | grep global-warroom | awk '{print $2}' | tail -n +2 | xargs kill -9

# 删除冲突的 Cron 文件
rm -f /etc/cron.d/ziwei-sync.disabled
```

**结果**:
```

✅ 全球战情室：1 个
```
