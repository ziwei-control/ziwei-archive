## 3. 创建第一个任务

### 方法 A: 使用模板（推荐）

1. 打开 Obsidian
2. 按 `Ctrl/Cmd + P` 打开命令面板
3. 输入 "Template" 选择 **Templates: Insert template**
4. 选择 **任务模板**
5. 填写任务信息

### 方法 B: 手动创建

1. 在 `docs/` 文件夹右键 → **新建笔记**
2. 命名为 `TASK-日期 - 序号_任务名称.md`
3. 复制任务模板内容
4. 填写任务详情

### 示例任务

已创建示例任务：
- **位置**: `/home/admin/Ziwei/data/tasks/inbox/TASK-20250227-001_测试任务.md`
- **内容**: 创建一个 Python 计算器
- **状态**: 等待处理

## 4. 任务工作流

```
Inbox (收件箱)
    ↓
T-01 首席架构师分解任务
    ↓
分发给对应岗位执行
    ↓
T-03 代码审计员审查
    ↓
交付 → 归档
```

### 任务状态

| 状态 | 说明 |
|------|------|
| **Inbox** | 新任务，等待处理 |
| **In Progress** | 进行中 |
| **Audit** | 审计中 |
| **Done** | 已完成 |
| **Archived** | 已归档 |

## 5. 查看任务进度

### 在驾驶舱查看

打开 `紫微智控驾驶舱.md`，可以看到：

- 系统状态
- 今日统计
- 进行中任务
- 最近沟通日志

### 使用 Dataview 查询

如果安装了 Dataview 插件，可以查询任务：

````markdown
```dataview
TABLE 优先级，截止时间，状态
FROM "tasks"
WHERE 状态 = "In Progress"
SORT 优先级 ASC
```
````

## 6. 推荐插件

### 必备插件

| 插件 | 用途 | 安装方法 |
|------|------|---------|
| **Templates** | 模板系统 | 核心插件，启用即可 |
| **Dataview** | 数据查询 | 社区插件 → 搜索 "Dataview" |
| **Full Calendar** | 日历视图 | 社区插件 → 搜索 "Full Calendar" |
| **Smart Connections** | 智能关联 | 社区插件 → 搜索 "Smart Connections" |

### 安装社区插件

1. 设置 → 第三方插件
2. 关闭"安全模式"
3. 浏览 → 搜索插件名称
4. 安装 → 启用

## 8. 双向链接

Obsidian 的强大功能：**双向链接**

### 创建链接

在笔记中使用 `[[链接目标]]`：

```markdown
这个任务由 [[T-01 首席架构师]] 分解。
参考文档：[[SOP/01_岗位说明书]]
```

### 查看关联

- 按 `Ctrl/Cmd + O` 快速跳转
- 右侧边栏 → 反向链接 → 查看谁链接了当前笔记

### 图谱视图

按 `Ctrl/Cmd + G` 打开图谱，可视化查看所有笔记的关联关系。
