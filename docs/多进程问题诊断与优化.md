## 📊 **问题诊断**

### **1. 自动同步监控 (25 个进程)**

**问题**: ❌ **严重 - 每小时重复启动**

**原因分析**:
```
01:18 - Supervisor 启动 (PID: 709198) ✅ 正常
01:30 - Cron 启动 (PID: 711232, 711234) ❌ 重复
02:00 - Cron 启动 (PID: 716473, 716476) ❌ 重复
02:30 - Cron 启动 (PID: 722013, 722014) ❌ 重复
03:00 - Cron 启动 (PID: 727217, 727220) ❌ 重复
...每 30 分钟重复一次
```

**根本原因**:
1. **Supervisor 配置** - 已配置为持续运行 ✅
2. **Cron 定时任务** - 每 30 分钟启动一次 ❌ 冲突
3. **旧进程未停止** - 新进程启动，旧进程还在运行 ❌

**资源浪费**:
- 内存：25 个进程 × 5MB = **125MB 浪费**
- CPU: 25 个进程轮流运行 = **资源浪费**

**原因分析**:


**原因**:
- 旧实例未停止
- Supervisor 启动新实例

## 🛠️ **优化方案**

### **方案 1: 停止 Cron 重复启动（推荐）**

**步骤**:

1. **检查 Cron 配置**
```bash
# 查看 cron 任务
crontab -l

# 或查看系统 cron
cat /etc/cron.d/* | grep -E "sync|warroom|log-trim"
```

2. **注释掉重复的 cron 任务**
```bash
# 编辑 cron
crontab -e

# 注释掉这些行（如果存在）
# */30 * * * * /home/admin/Ziwei/scripts/auto-sync-watchdog.py
# */60 * * * * /home/admin/Ziwei/scripts/log-trim.py
# */30 * * * * /home/admin/Ziwei/scripts/global-warroom-upgraded.py
```

3. **清理旧进程**
```bash
# 停止所有重复进程
pkill -f "auto-sync-watchdog"
pkill -f "log-trim.py"
pkill -f "global-warroom-upgraded"

# 让 Supervisor 重新启动（只启动 1 个）
supervisorctl restart ziwei-auto-sync
supervisorctl restart ziwei-log-trim
supervisorctl restart ziwei-warroom
```

4. **验证结果**
```bash
# 检查进程数
ps aux | grep auto-sync | grep -v grep | wc -l  # 应该 = 1
ps aux | grep log-trim | grep -v grep | wc -l   # 应该 = 1
ps aux | grep warroom | grep -v grep | wc -l    # 应该 = 1
```

### **方案 2: 修改 Supervisor 配置（如果确实需要 Cron）**

**如果 Cron 是必要的**（比如需要定时重启），修改 Supervisor 配置：

```ini
# /etc/supervisor/conf.d/ziwei.conf
```

**然后只用 Cron**，但这不是最佳实践。

## ⚠️ **注意事项**

### **1. 不要同时使用 Supervisor 和 Cron**

**错误做法**:
```bash
# Supervisor 配置持续运行
[program:ziwei-auto-sync]
autostart=true

# Cron 又定时启动
*/30 * * * * python3 auto-sync-watchdog.py
```

**正确做法**:
```bash
# 只用 Supervisor
[program:ziwei-auto-sync]
autostart=true
autorestart=true

# Cron 只用于日志清理等定时任务
0 0 * * * find /var/log -name "*.log" -mtime +7 -delete
```

### **2. 监控进程数**

```bash
# 创建监控脚本
cat > /home/admin/Ziwei/scripts/check_process_count.sh << 'EOF'
#!/bin/bash
echo "进程数检查:"
echo "自动同步：$(ps aux | grep auto-sync | grep -v grep | wc -l) (应该=1)"
echo "日志修剪：$(ps aux | grep log-trim | grep -v grep | wc -l) (应该=1)"
echo "全球战情室：$(ps aux | grep warroom | grep -v grep | wc -l) (应该=1)"
EOF

chmod +x /home/admin/Ziwei/scripts/check_process_count.sh

# 定期运行
./scripts/check_process_count.sh
```
