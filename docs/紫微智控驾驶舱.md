## 今日统计

```dataview
TABLE file.ctime as 创建时间, file.outlinks as 关联
FROM "tasks"
WHERE date(file.day) = date(today)
SORT file.ctime DESC
```

---

## 进行中任务

```dataview
TABLE 优先级，截止时间，状态
FROM "tasks"
WHERE 状态 = "In Progress"
SORT 优先级 ASC
```

## 最近沟通日志
