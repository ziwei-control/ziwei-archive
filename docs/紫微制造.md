## 🚀 快速开始

```bash
# 1. 创建任务说明书
mkdir -p tasks/MY-PROJECT
cat > tasks/MY-PROJECT/spec.md << 'EOF'
# 我的项目
## 任务描述
[详细描述]
## 功能需求
[功能列表]
EOF

# 2. 运行紫微制造
python3 scripts/ultimate_wisdom_creator.py tasks/MY-PROJECT/spec.md

# 3. 获得完整项目
# 项目在 tasks/MY-PROJECT/ 目录
```
