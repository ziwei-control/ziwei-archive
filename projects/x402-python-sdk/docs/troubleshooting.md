# 🔧 故障排查指南

## 常见问题

### 连接问题

#### Q: 连接失败，提示 404 Not Found

**解决方案**:
1. 检查 API 基础 URL 是否正确
2. 确认服务是否运行
3. 检查网络连接

#### Q: 超时错误

**解决方案**:
1. 增加 timeout 参数（默认 30 秒）
2. 检查网络延迟
3. 尝试重试

---

### 支付问题

#### Q: 支付失败

**解决方案**:
1. 检查钱包 USDC 余额
2. 检查 Base 链 gas 费
3. 检查支付证明格式

#### Q: 支付验证失败

**解决方案**:
1. 检查交易哈希是否正确
2. 确认交易已确认
3. 检查收款地址是否正确

---

### 代码问题

#### Q: 如何处理异步调用？

A: SDK 不支持异步，需要自己实现异步逻辑

#### Q: 如何批量调用？

A: 参考 [高级示例](examples/advanced_usage.py) 的批量调用示例
