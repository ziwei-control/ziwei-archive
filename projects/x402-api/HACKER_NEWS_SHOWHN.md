**Q: Why USDC only? Why not credit card?**
A: Credit card requires Stripe/PayPal integration, which takes time. USDC allows global access without geographic restrictions. We're working on adding PayPal in the next update.

**Q: Is this sustainable at $0.02/call?**
A: Yes. Our cost is ~$0.01/call (Aliyun Bailian API), so we have ~50% margin. At scale, we can negotiate better rates.

**Q: What happens if you shut down?**
A: Good question. We're building in public and open-sourcing parts of the code. Users can self-host if needed.

**Q: How is this different from RapidAPI?**
A: RapidAPI is a marketplace. We're the actual API provider with our own payment protocol (x402).

**Q: Why should I trust you?**
A: Don't trust us, trust the code. Everything is on GitHub. Try the free trial. We're building in public with transparent revenue.
