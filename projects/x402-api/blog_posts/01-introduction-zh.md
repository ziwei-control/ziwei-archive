## 💡 解决方案：x402 微付费协议

x402 是一个基于 Web3 的支付协议，让 AI API 可以：

- ✅ **按次付费** - 用一次付一次，不用不花钱
- ✅ **超低门槛** - 单次调用最低 $0.02 USDC
- ✅ **无需注册** - 有 USDC 钱包就能用
- ✅ **即时结算** - 支付完成立即获得 API 响应
- ✅ **全球可用** - 不受信用卡、地域限制

## 🚀 5 分钟快速开始

### 第一步：准备 USDC 钱包

安装 MetaMask 或任何支持 Base 链的钱包，充值少量 USDC（$5 就够用很久）。

### 第二步：调用 API

```python
import requests

# API 端点（合作后获取）
url = "https://api.x402.network/api/v1/translator"  # 示例地址

# 请求数据
payload = {
    "text": "Hello, world!",
    "source": "en",
    "target": "zh"
}

# 添加 x402 支付头（示例，实际需要用 x402 库）
headers = {
    "X-Payment-Amount": "20000",  # 0.02 USDC (6 位小数)
    "X-Payment-Token": "USDC",
    "X-Payment-Signature": "your_signature_here"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```

### 第三步：看到结果

```json
{
  "success": true,
  "data": {
    "translated_text": "你好，世界！",
    "source_language": "en",
    "target_language": "zh"
  },
  "cost": "0.02 USDC"
}
```

**完成！** 你刚刚用 $0.02 完成了一次翻译。

## ❓ 常见问题

**Q: USDC 是什么？**  
A: USDC 是一种稳定币，1 USDC ≈ 1 美元，由 Circle 公司发行，受美国监管。

**Q: 我没有 USDC 怎么办？**  
A: 在币安、Coinbase 等交易所购买，然后提现到钱包。

**Q: 支付安全吗？**  
A: x402 是开源协议，支付在链上完成，我们拿不到你的私钥。

**Q: 可以退款吗？**  
A: 区块链支付不可逆，但如果 API 有问题，联系我们补偿。

**Q: 有发票吗？**  
A: 目前不提供发票（去中心化支付），企业用户可联系定制方案。
