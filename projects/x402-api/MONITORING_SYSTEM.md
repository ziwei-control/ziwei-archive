## 🤖 自动化功能

### 1. 服务状态监控

**频率：** 每分钟检查一次

**监控内容：**
- ✅ x402 API (端口 5002)
- ✅ API Key Server (端口 8090)
- ✅ User Dashboard (端口 8091)
- ✅ Main Dashboard (端口 8081)
- ✅ Nginx (端口 80)

**自动修复：**
- 发现服务未运行 → 自动重启
- 重启后验证 → 确认成功
- 记录日志 → 便于追踪

---

### 2. 错误日志管理

**自动清空：**
- 每天自动清空错误日志
- 避免日志文件过大
- 保留最新错误信息

**磁盘清理：**
- 磁盘使用率 > 90% 时触发
- 清理 7 天前的日志文件
- 释放磁盘空间

### 3. 系统资源监控

**磁盘监控：**
- 阈值：90%
- 自动清理旧日志
- 记录使用率

**内存监控：**
- 阈值：90%
- 发出告警
- 记录使用率
