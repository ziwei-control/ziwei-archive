### 解决方案（2:00-4:00）

```
[画面：x402 协议架构图]
x402 是一个基于 Web3 的支付协议，让 AI API 可以：

✅ 按次付费 - 用一次付一次，不用不花钱
✅ 超低门槛 - 单次调用最低 $0.02 USDC
✅ 无需注册 - 有 USDC 钱包就能用
✅ 即时结算 - 支付完成立即获得 API 响应
✅ 全球可用 - 不受信用卡、地域限制
```

```

import requests

url = "http://8.213.149.224:5002/api/v1/translator"
payload = {"text": "Hello!", "source": "en", "target": "zh"}
headers = {
    "X-Payment-Amount": "20000",
    "X-Payment-Token": "USDC"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```
