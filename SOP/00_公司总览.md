## 核心架构

### 运行环境

```
┌─────────────────────────────────────────────────────────┐
│                    云端（大脑）                          │
│              阿里百炼平台 - 所有 AI 推理任务                │
│   Qwen3-Max / Qwen3-Coder / GLM4.7 / Kimi2.5 / Vision  │
└─────────────────────────────────────────────────────────┘
                          ↕ API 调用
┌─────────────────────────────────────────────────────────┐
│                    本地（躯干）                          │
│            2G 内存云主机 - 轻量调度，不做算力支出           │
│   Python 脚本 + SQLite + Git + Obsidian（纯文本存储）    │
└─────────────────────────────────────────────────────────┘
```

### 架构原则

| 原则 | 说明 |
|------|------|
| **云地分离** | 云端算力，本地管控 |
| **脚本为王** | 纯 Python 脚本，无 Docker 依赖 |
| **异步解耦** | 文件系统作为任务队列，避免阻塞 |
| **串行执行** | 2G 内存限制，任务串行处理 |
| **全平台兼容** | Windows/macOS/Linux/树莓派均可部署 |

### 战损与荣耀档案库

```

├── SUCCESS/PENDING_RELEASE/  # 高分待发布项目（可变现）
├── SUCCESS/ARCHIVE/          # 历史成功案例
├── FAILURE/                   # 失败案例（避坑指南）
└── ERROR/                     # 系统错误（技术复盘）
```
